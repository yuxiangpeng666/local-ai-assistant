# Local AI Assistant

## Overview
This project is a Python-based local AI assistant designed to support conversational interaction through a simple chat interface.

## Features
- Built with Python
- Supports basic conversational logic
- Includes a simple chat UI
- Explores local AI assistant workflow and user interaction design

## Technologies Used
- Python
- JSON
- Tkinter / Python GUI

## Purpose
This project was developed to strengthen my practical skills in Python programming, user interaction design, and AI-related application development.

## Future Improvements
- Improve response logic
- Enhance interface design
- Add more intelligent conversation handling

## How to Run
1. Make sure Python is installed.
2. Download the project files.
3. Run the chat_ui.py file to start the interface.

## Project Structure
- ai_core.py: core assistant logic
- chat_ui.py: graphical chat interface
