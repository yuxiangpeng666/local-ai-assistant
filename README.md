# Local AI Assistant

## Overview
This project is a Python-based local AI assistant designed to support conversational interaction through a simple chat interface.

## Features
- Built with Python
- Supports basic conversational logic
- Includes a simple chat UI
- Explores local AI assistant workflow and user interaction design

## Technologies Used
- Python
- JSON
- Tkinter / Python GUI

## Purpose
This project was developed to strengthen my practical skills in Python programming, user interaction design, and AI-related application development.
